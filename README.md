# 🌐 Global Job Hub

A high-performance, automated job search platform providing real-time access to career opportunities worldwide. This project integrates live data from the **Jooble API** to deliver a seamless search experience for job seekers across all industries.

## 🚀 Live Demo
You can view the live website here: [https://global-job-hub.github.io/](https://global-job-hub.github.io/)

## ✨ Key Features
* **Real-Time Listings:** Fetches the latest job openings globally using REST API integration.
* **SEO Optimized:** Implements JSON-LD structured data to ensure listings are discoverable by major search engines.
* **Clean URL Structure:** Designed for high readability and professional indexing (e.g., `/jobs/software-engineer`).
* **Fully Responsive:** Optimized UI for mobile, tablet, and desktop browsers.
* **Verification Ready:** Includes standard root-level support for domain ownership and security verification files.

## 🛠️ Technical Implementation
* **Language:** HTML5, CSS3, JavaScript (ES6+).
* **API Integration:** Connects to the Jooble Search API for dynamic content delivery.
* **Hosting:** Served via GitHub Pages for high availability.
* **Architecture:** Static site generation with client-side dynamic fetching.

## 📁 Project Structure
* `/index.html`: Main landing page and search interface.
* `/jobs/`: Dedicated directory for individual career listings.
* `/assets/`: Styling and scripting resources.
* `/ads.txt`: Project sustainability and verification credentials.

## ⚖️ Project Sustainability
To keep this service free and updated for users, the project may include links to partners and supported organizations. These help cover the API usage costs and ongoing development of the platform.
